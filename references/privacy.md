# Privacy Reference

## Local-first Defaults

Keep resumes, job files, and preferences local by default. A request to compare or investigate named companies authorizes research of public company information with Codex's built-in web tools. Ask separately before sending private candidate material to any external service.

## Do Not Request

- passwords
- cookies
- access tokens
- private chat logs
- work source code
- internal datasets
- production logs
- model weights
- unreleased algorithms
- client confidential information

## Disclosure Handling

- `public`: may be used in reports.
- `abstract_only`: describe capability without naming private systems, clients, metrics, or code.
- `private`: may inform reasoning but should not appear in public reports.
- `disallowed`: must not be used.

## Company Risk Statements

Negative company conclusions require source URL, title, access date, and careful wording. If sources are weak, mark uncertainty instead of inventing facts.
